About

This website generates a Fibonacci Sequence with a specified number of elements.

The sequence is generated by a recursive function as seen below:

      const fibonacci = (i, sequence = []) =>
        i === 0
          ? sequence 
          : sequence.length === 0  
          ? fibonacci(i - 1, [0])
          : sequence.length === 1
          ? fibonacci(i - 1, [0, 1])
          : fibonacci(i - 1, [
              ...sequence,
              sequence[sequence.length - 1]
               + sequence[sequence.length - 2],
            ]);

This website generates an array with a specified number of random number elements, which it then sorts using a Merge Sort .

The sequence is generated by a recursive function as seen below:

      const sort = (arrayOne, arrayTwo) => {
        const collector = [];

        while (arrayOne.length > 0 && arrayTwo.length > 0) {
          arrayOne[0] < arrayTwo[0]
            ? collector.push(arrayOne.shift() as number)
            : collector.push(arrayTwo.shift() as number);
        }
      };

      const mergeSort = (array) => {
        const middle = Math.floor(array.length / 2);

        return array.length <= 1
          ? array
          : sort(mergeSort(array.slice(0, middle)),
           mergeSort(array.slice(middle)));
      };
